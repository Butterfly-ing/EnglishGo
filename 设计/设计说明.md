# EnglishGo 英文学习平台 - 设计说明

## 项目概述
基于 Django + Vue3 的全栈英文学习平台，覆盖初中→SAT 七个阶段约 20,000+ 单词，支持闪卡记忆、三种测验模式、进度追踪。

## 技术栈
| 层 | 技术 | 备注 |
|-----|------|------|
| 后端 | Python 3 + Django 4.2 + DRF | JWT 认证，CORS 跨域 |
| 前端 | Vue 3 (Composition API) + Vue Router + Pinia + Axios | SPA 单页应用 |
| 数据库 | **MySQL 8**（pymysql 驱动） | 数据库名 `englishgo`，utf8mb4 |
| 主题 | CSS Variables，5 套风格 × 深/浅色 | 10 套主题一键切换 |

## 核心数据流

```
db/*.sql (原始数据, ~5MB)
    │
    ▼ python manage.py import_exam_words
    ├── [0/5] 读取 SQL → 建 MySQL 源表 (junior/senior/CET4/...)
    ├── [1/5] 读取 7 源表原始数据
    ├── [2/5] 清洗 translate 字段（去语法冗余）+ 跨表去重
    ├── [3/5] 按考试分类统计
    ├── [4/5] 跳过 word 表已有词
    └── [5/5] bulk_create 批量写入 word 表
    │
    ▼
word 表 (~15,000+ 条去重后)
    │
    ▼ API: GET /api/v1/vocabulary/words/?category=cet4
    ▼ 前端各页面按 category 筛选展示
```

### 清洗逻辑示例

```
原始 translate: "n.政党，党派；聚会，派对[复数parties]v.参加社交聚会[过去式partied过去分词partied现在分词partying]"
        ↓ 提取第一个词性 → POS = noun
        ↓ 去掉 [复数xxx] [过去式xxx] 等语法冗余
        ↓ 遇到第二个词性标记 v. 时截断
        ↓
清洗结果:
  chinese:         "政党，党派；聚会，派对"
  part_of_speech:  "noun"
```

## 功能模块

### 1. 用户模块 (users)
- 注册 / 登录 / 退出
- JWT 令牌认证（有效期 12h，支持刷新）
- 个人学习统计

### 2. 词汇模块 (vocabulary) ★
- 单词 **20,000+**（7 个考试覆盖）
- 按**考试分类筛选**：初中 / 高中 / 四级 / 六级 / 考研 / 托福 / SAT
- 按难度分类筛选：基础 / 中级 / 高级 / 商务 / 旅游
- 单词搜索（英文/中文）
- 收藏 & 掌握标记
- 单词卡片显示：英文、音标、中文、**考试标签（如 🏷四级）**、词性、难度星级、例句

### 3. 测验模块 (quiz)
- 选择题测验（英译中 / 中译英）
- 听写模式（显示英文，输入中文）
- 拼写模式（显示中文，输入英文）
- **支持按考试分类出题**（如只测"四级"词）
- 测验结果统计
- ✅ 选择题选项在深色模式下可正常阅读（已修复硬编码颜色）

### 4. 闪卡学习
- 看英文想中文 / 看中文想英文 / 混合模式
- **卡片左上角显示考试来源标签**（如"四级"、"托福"）
- 忘记了 → 加入复习队列

### 5. 学习进度模块 (progress)
- 每日学习打卡（连续天数）
- 学习时长统计
- 正确率统计
- 学习趋势图表

### 6. 文章阅读模块 (articles)
- 分级阅读（入门/中级/高级）
- 中英对照
- 阅读计数

## 数据库设计

### Word 表（核心，~15,000+ 条去重后）

| 字段 | 类型 | 说明 |
|------|------|------|
| id | Integer | 主键 |
| english | String(200) | 英文单词 |
| chinese | String(200) | 中文释义（清洗后，最大 200 字符） |
| phonetic | String(200) | 音标（可后续 API 批量补充） |
| part_of_speech | String(20) | 词性：noun/verb/adjective/adverb/preposition/conjunction/pronoun/phrase |
| example | Text | 例句（可后续 API 批量补充） |
| **category** | **String(20)** | **考试分类：junior/senior/cet4/cet6/postgraduate/toefl/sat + basic/intermediate/advanced/business/travel** |
| difficulty | Integer(1-5) | 难度：初中=1，高中=2，四级=3，六级/考研=4，托福/SAT=5 |
| created_at | DateTime | 创建时间 |

> **设计说明**：使用单表 + `category` 字段区分考试，而非 7 张独立表。优点：
> - API 原生支持 `?category=cet4` 筛选
> - 跨考试去重（同一单词出现在多个级别时，保留最高难度）
> - 无数据冗余，CRUD 统一

### 7 个考试源表（仅作原始数据存储，导入后不再使用）

| 表名 | 考试 | 原始数量 | 对应 SQL |
|------|------|---------|----------|
| `junior` | 初中 | ~3,400 | `1 初中-乱序_sql.sql` |
| `senior` | 高中 | ~5,900 | `2 高中-乱序_sql.sql` |
| `CET4` | 四级 | ~6,100 | `3 四级-乱序_sql.sql` |
| `CET6` | 六级 | ~4,700 | `4 六级-乱序_sql.sql` |
| `graduate` | 考研 | ~9,600 | `5 考研-乱序_sql.sql` |
| `TOEFL` | 托福 | ~10,900 | `6 托福-乱序_sql.sql` |
| `SAT` | SAT | ~7,900 | `7 SAT-乱序_sql.sql` |

## 主题系统

支持 10 套主题（5 风格 × 深/浅色），通过 CSS Variables 全局切换：

| 风格 | 深色变量 | 浅色变量 |
|------|----------|----------|
| 未来风 (Future) | `theme-future-dark` | `theme-future-light` |
| 典雅风 (Elegant) | `theme-elegant-dark` | `theme-elegant-light` |
| 护眼风 (Nature) | `theme-nature-dark` | `theme-nature-light` |
| 极简风 (Minimal) | `theme-minimal-dark` | `theme-minimal-light` |
| 活力风 (Vibrant) | `theme-vibrant-dark` | `theme-vibrant-light` |

所有业务组件使用 `var(--text)` `var(--bg-card)` 等变量，自动跟随主题切换。

## 启动方式

```bash
# 1. 后端
cd backend
conda activate <你的环境>
pip install -r requirements.txt
python manage.py migrate
python manage.py import_exam_words    # ★ 只跑一次，导入全部单词
python manage.py runserver 0.0.0.0:8000

# 2. 前端
cd frontend
npm install
npm run dev                           # 默认 http://localhost:3000
```

或直接双击 `启动后端.bat` / `启动前端.bat`。
